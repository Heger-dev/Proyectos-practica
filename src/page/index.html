Codificacion y descodificacion de texto

Como fue creado (HTML, CSS, JS)

Se utiliza un array tipo alfabeto y se cambia el texto al numero del indice del caracter dentro del array y viceversa. Los numeros menores a 9 tendran un 0 antes del numero para que todos los numeros tengan 2 digitos.
Tambien para el hecho de que se ponga el traduciendo se utiliza un setTimeout para que durante 800ms aparezca traduciendo para luego ejecutarse la traduccion.
