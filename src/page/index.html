An alphabet type array is used and the text is changed to the index number of the character within the array and vice versa. Numbers less than 9 will have a 0 before the number so that all numbers have 2 digits.
Also for the fact that the translating is set, a setTimeout is used so that for 800ms it appears translating and then the translation is executed.
